# MIT OpenCourseWare

## Overview
MIT publishes material from over 2500 courses on <a href="https://ocw.mit.edu/" target="_blank">OpenCourseWare</a>. These come from a very large breadth of subjects (basically anything that MIT offers). For PSU CS students specifically, MIT offers several useful courses in AI, Data Science, and computer systems. These courses will help you gain valuable skills that will help you in the workforce. As MIT courses, these also go more in depth than PSU courses. 

## Using the Material
I recommend treating each course you choose to take like a real course. This means you should watch all of the video lectures and complete all of the coding exercises they offer. I also recommend studying for and taking the tests they offer. While you won’t actually get credit from this, it will help you assess where you’re knowledgeable and where you should try to learn more. 

## Suggested Courses
The following are courses that I recommend taking if you want to become more knowledgeable in AI engineering / machine learning. If you don’t plan to specialize in machine learning, I recommend using AI to create a plan of courses for yourself based on your goals.
- <a href="https://ocw.mit.edu/courses/6-s191-introduction-to-deep-learning-january-iap-2020/" target="_blank">6.S191 (Introduction to Deep Learning)</a> - Easiest deep learning course, explains the bigger picture for someone with little ML knowledge. If you have significant ML knowledge, skip this course.
- <a href="https://openlearninglibrary.mit.edu/courses/course-v1:MITx+6.036+1T2019/about" target="_blank">6.036 (Introduction to Machine Learning)</a> - This one provides more depth and rigor in machine learning.
- <a href="https://ocw.mit.edu/courses/6-824-distributed-computer-systems-engineering-spring-2006/" target="_blank">6.824 (Distributed Systems)</a> - While this one isn’t directly AI oriented, it is still very important in helping you learn how to improve AI when used at scale.
- <a href="https://ocw.mit.edu/courses/18-06ci-linear-algebra-communications-intensive-spring-2004/" target="_blank">18.06 (Linear Algebra)</a> - Linear Algebra is very important for Machine Learning, but I would only recommend using this when necessary since Math 220 is required at PSU and will likely be sufficient for most purposes.

## Tips
- If you’re taking a PSU course, but want more breadth, it may be worth it to check out a related MIT course.
- <a href="https://see.stanford.edu/" target="_blank">Stanford</a> also offers resources of a similar quality.
